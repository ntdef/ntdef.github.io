Puzzzzzlss!! Submission to PuzzlOR Feb 2015

Date Sat 14 March 2015 Tags puzzles / combinatorics

Sometimes in a moment of weakness I'll wander about to math/programming puzzle page and try my luck at the newest posting. I've been slogging through the Project Euler problems in python, but every now and then I'll check PuzzlOR, a website dedicated to puzzles relating to operational research and logistics. The puzzles there range from medium difficulty to almost trivial (i.e. you don't really need a computer to solve the puzzle).

This month's problem lies among the trivial. But hey, for the chance to win $25? Heck yeah. Basically, the question is how do you order a set of 3 pennies, 3 nickels and 3 dimes such that they sum across the rows and columns equal to the values in the pictured below.

In [1]: 
import numpy as np
import itertools as it
from math import factorial
PNNY, NCKL, DIME = 1, 5, 10
COINS = [PNNY, NCKL, DIME]
ROW_ANS = np.array([11, 12, 25])
COL_ANS = np.array([21, 16, 11])

How many permutations do we need to try?

The first question is, how difficult is this problem going to be to compute? As it turns out, not very: to compute the number of $n$ permutations of $k$ items, we have

$$ \frac{n!}{n_1!n_2!n_3!...n_k!} $$

In this case, we have 3 distinct items, each with 3 elements. So the total possibility space is $\frac{9!}{3!^3} = 1680$ permutations. We can verify that this is true by generating all possibilities.

In [2]: 
def generate_permutations(): return list(set(it.permutations(COINS*3)))

print "How many permutations?"
print len(generate_permutations())

How many permutations?
1680

Since the number of things we need to try are so few, we can just brute force our way to a solution. Now to find our solution, we'll turn every permutation into a matrix and search the collection of matrices by summing rows and columns until we find a matrix that meets our criteria.

In [13]: 
def mk_matrix_from_list(ls):
    return np.array(ls).reshape(3,3)
def row_sums(mat):
    return mat.sum(axis=1)
def col_sums(mat):
    return mat.sum(axis=0)
In [14]: 
print "\nAnd the answer is ... \n "
for i in map(mk_matrix_from_list, generate_permutations()):
    if np.array_equal(row_sums(i), ROW_ANS) and np.array_equal(col_sums(i), COL_ANS):
        print i


And the answer is ... 
 
[[ 1  5  5]
 [10  1  1]
 [10 10  5]]

And we're done!
